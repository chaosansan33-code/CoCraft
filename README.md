# 記藝共養所網站

## GitHub Pages 上傳方式
請把整個資料夾內容上傳到 GitHub repository 根目錄。

必須包含：
- index.html
- products.html
- product-umbrella.html
- product-bamboo.html
- product-chunzihua.html
- plans.html
- nfc.html
- contact.html
- style.css
- script.js
- images/ 資料夾

## 圖片檔名
請把圖片放在 images 資料夾，並依照以下檔名命名：

- hero.jpg：首頁滿版職人主視覺
- story.jpg：品牌故事圖片
- nfc-demo.jpg：NFC 感應示意圖
- umbrella.jpg：美濃油紙傘商品卡圖片
- bamboo.jpg：竹籐編織商品卡圖片
- chunzihua.jpg：春仔花商品卡圖片
- umbrella-detail.jpg：美濃油紙傘詳細頁大圖
- bamboo-detail.jpg：竹籐編織詳細頁大圖
- chunzihua-detail.jpg：春仔花詳細頁大圖

## 注意
檔名大小寫必須完全一致。例如 hero.jpg 不能叫 Hero.JPG。
